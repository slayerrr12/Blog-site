What is Node.js?

Node.js is an open-source, cross-platform JavaScript runtime environment that executes JavaScript code outside of a web browser. It allows developers to build server-side applications with JavaScript, which was traditionally used only for client-side scripting in web browsers.

Node.js uses an event-driven, non-blocking I/O model that makes it lightweight and efficient. It includes a built-in package manager called npm, which allows developers to easily share and reuse code.

Node.js Syntax

Node.js syntax is similar to other programming languages, such as JavaScript, Java, and C++. It includes statements, variables, functions, and expressions.

The complete code for this example would look like:

      const http = require('http');
const hostname = '127.0.0.1';
const port = 3000;
const server = http.createServer((req, res) => {
res.statusCode = 200;
res.setHeader('Content-Type', 'text/plain');
res.end('Hello, World!\n');
});
server.listen(port, hostname, () => {
console.log(Server running at http://${hostname}:${port}/);
});

Explaination

  • Create a Node.js HTTP server:
    • Require the built-in http module with the following code: const http = require('http');
    • Define the hostname and port number for the server with the following code: const hostname = '127.0.0.1'; and const port = 3000;
    • Create the server with the following code: 
              
const server = http.createServer((req, res) => {
  res.statusCode = 200;
  res.setHeader('Content-Type', 'text/plain');
  res.end('Hello, World!\n');
});
      This code creates an HTTP server that listens for incoming requests on the specified hostname and port. When a request is received, the server sends a response with a status code of 200 and a message of "Hello, World!".
  • Start the server with the following code: 
          
server.listen(port, hostname, () => {
  console.log(`Server running at http://${hostname}:${port}/`);
});
    This code tells the server to start listening for incoming requests on the specified hostname and port. When the server starts successfully, it logs a message to the console with the URL where the server is running.

Node.js Features

  • Node.js has a built-in module called HTTP, which allows Node.js to transfer data over the Hyper Text Transfer Protocol (HTTP).

  • Node.js has a built-in module called File System, which allows you to work with the file system on your computer.

  • Asynchronous and event-driven
  • Fast performance
  • Scalable
  • Runs on multiple platforms
  • Large and active developer community
  • Allows for easy sharing and reuse of code through npm
